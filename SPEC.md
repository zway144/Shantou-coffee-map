# 汕头咖啡店分布地图网站 - 规格文档

## 1. 项目概述

- **项目名称**: 汕头咖啡地图 (Shantou Coffee Map)
- **项目类型**: 单页面交互式地图网站
- **核心功能**: 在百度地图上展示汕头24家咖啡店的位置，支持hover查看详情
- **目标用户**: 咖啡爱好者、游客、本地居民

## 2. UI/UX 规格

### 布局结构

- **整体布局**: 全屏地图 + 顶部悬浮标题栏 + 右侧悬浮信息卡片
- **顶部标题栏**: 固定在顶部，高度60px，半透明毛玻璃效果
- **地图区域**: 占满整个视口
- **右侧信息卡片**: 悬浮在地图右侧，宽度320px，动态展示hover的店铺信息

### 响应式断点

- 桌面端: > 1024px (完整布局)
- 平板端: 768px - 1024px (信息卡片可收起)
- 移动端: < 768px (底部弹出式信息栏)

  - 副标题: 14px, font-weight: 400
  - 卡片标题: 18px, font-weight: 600
  - 卡片正文: 14px, font-weight: 400

- 基础单位: 8px

- 背景: 半透明白色毛玻璃 `rgba(255,255,255,0.85)`
- 左侧: 主标题 "汕头咖啡地图" + 副标题 "探索城市中的咖啡香"
- 右侧: 咖啡杯图标 + 店铺数量 "24家"

#### 2. 地图标记 (自定义Marker)

- 选中状态: 金色边框 + 详细信息

  - 装饰性分隔线

### 动画效果

- **页面加载**: 标题栏从顶部滑入 (0.5s ease-out)
- **卡片出现**: 右侧卡片从右侧滑入 (0.3s ease-out)
- **Marker hover**: 轻微上浮 + 放大 (0.2s ease)
- **信息切换**: 渐隐渐现 (0.25s ease-in-out)
- **背景纹理**: 极细微的噪点纹理，增加纸张质感

## 3. 功能规格

### 核心功能

1. **地图展示**
   - 初始化定位到汕头市区中心 (经度: 116.68, 纬度: 23.35)
   - 默认缩放级别: 13
   - 支持缩放、平移

2. **咖啡店标记**
   - 24个自定义Marker
   - 点击/hover显示信息
   - Marker使用自定义图标（咖啡杯SVG）

3. **信息展示**
   - 右侧卡片实时显示hover的店铺信息
   - 显示店铺名称、详细地址

4. **交互行为**
   - 鼠标悬停在Marker上 → 显示信息卡片
   - 鼠标移出 → 延迟500ms隐藏信息卡片
   - 点击Marker → 固定显示信息卡片

### 百度地图API集成

- 使用百度地图JavaScript API v3.0
- 需要AK密钥 (用户需提供或使用演示密钥)
- 加载自定义样式: 简约灰度风格

### 数据处理

- 咖啡店数据从Excel读取，包含：
  - 店铺名称
  - 地址
  - 经纬度 (需要通过地址解析获取)
